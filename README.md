# Paycer Prototype

The Paycer Prototype repository contains an MVP of the decentralized Paycer platform.

# Requirements
- NodeJS
- Yarn
- Git


# Obtain RPC URLs and add them in dotenv 
- https://www.alchemy.com/

# Getting started
```
# install depedencies
yarn install

# start developing
yarn dev

# start developing isolated components in the styleguide
yarn storybook
```
